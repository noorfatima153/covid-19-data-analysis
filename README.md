# covid-19-data-analysis
Big Data analysis of COVID-19 dataset using PySpark with filtering, aggregation, and insights.

This project performs COVID dataset analysis using PySpark.

## Tasks Performed
- Load COVID dataset
- Filter country-specific data
- Find total cases and deaths
- Sort by highest cases

## Technologies
- Python
- PySpark
- Pandas

## Dataset
COVID CSV dataset

## How to Run
Install requirements and run the notebook.
